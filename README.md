# Launchpad (Crypto Token Launcher)

A crypto token launchpad (simulation) website built with Java/Jakarta Server Pages as the part of Web Programming course assignment (Quiz 2).

## Tech Stack
- **Backend:** Java, Jakarta Servlet, Maven
- **Frontend:** Tailwind 4
- **Database:** PostgreSQL
- **Web Server:** Apache Tomcat

## Learning Outcomes
- Create a dynamic website using JSP (Java/Jakarta Server Pages).
- Understand how to deploy using web server and docker.
- Understanding API integration.
